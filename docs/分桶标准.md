# 分桶标准

第一阶段采用保守分桶，优先保证可复核性，不追求高自动化。

## 1. 空白样本
满足以下任一条件：

- `原文` 为空
- `清洗后文本` 为空

## 2. 疑难样本
满足以下任一条件：

- 轻预洗阶段已打上 `命中疑难模式`
- 含有明显噪声片段，如 `96px`、`|class=`、`<ref`
- 无法稳定判断是否为正常 LaTeX 语料

## 3. LaTeX样本
在不命中空白样本和疑难样本前提下，文本包含明显 LaTeX 标记，例如：

- `\command`
- `$...$`
- `\begin{...}`
- `^` 或 `_` 等公式特征

## 4. 普通文本
在不命中上述三类时，统一归入普通文本。

## 执行原则

- 不确定就打标，不强判。
- 先小样本验证分桶结果，再考虑扩规则。
- 本仓库当前不处理脱壳和中文金标转换。
