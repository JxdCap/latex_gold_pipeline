# Documentation

本目录描述中文 LaTeX 语料清洗与分桶项目的第一阶段约束。

建议阅读顺序：

1. [项目目标](./项目目标.md)
2. [字段规范](./字段规范.md)
3. [分桶标准](./分桶标准.md)

当前实现边界：

- 只支持从 `data/输入表格/input.xlsx` 导入小样本
- 默认读取第一个工作表，A 列从第 2 行开始作为原文本
- 导入阶段的基准文件是 `data/原始数据/原始样本.jsonl`
- 只覆盖 `导入 -> 轻预洗 -> 分桶 -> 分流 -> 导出`
- 每阶段默认同时输出 `jsonl` 和 `xlsx`
- 不扩展到脱壳、中文金标转换、复杂批次特判、自动猜测公式含义

主要代码入口：

- `src/主流程.py`
- `src/导入器.py`
- `src/轻预洗.py`
- `src/分桶器.py`
- `src/分流器.py`
- `src/导出表格.py`
