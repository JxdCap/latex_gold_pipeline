# 项目目标

本项目用于中文 LaTeX 语料清洗与分桶。当前只实现第一阶段最小闭环，不扩展到脱壳、中文金标转换、复杂批次特判或公式语义猜测。

第一阶段目标如下：

1. 从 `data/输入表格/input.xlsx` 读取第一个 sheet 的 A 列，从第 2 行开始。
2. 导入器输出固定原始样本文件 `data/原始数据/原始样本.jsonl`，后续处理都从这个 `jsonl` 继续。
3. 导入器只做格式标准化，不做重清洗、不做分桶。
4. 先导入为原始样本，再做轻预洗，再做分桶，再做分流。
5. 每个阶段默认同时输出 `jsonl` 和 `xlsx`。
6. 先只支持小样本运行，不提供全量模式。
7. 无法稳定判断的样本统一打标并归入疑难桶。
8. 分流只按分桶结果和复核标记执行保守分流，不做复杂特判。

当前阶段的工作顺序固定为：

`导入 -> 轻预洗 -> 分桶 -> 分流 -> 导出`

本阶段不包含：

- 脱壳
- 中文金标转换
- 复杂批次特判
- 自动猜测公式含义
